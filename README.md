🌱 AgroGuide – Smart Agriculture Web App

📌 Description
AgroGuide is an AI-powered agriculture web application designed to assist farmers with intelligent crop recommendations, weather insights, and farming guidance. The system uses machine learning models and APIs to provide accurate and real-time suggestions.

---

🚀 Features
- 🌾 AI-based crop recommendation system  
- 🌦️ Weather insights for better decision-making  
- 🤖 Smart farming guidance using NLP models  
- 💻 User-friendly web interface  

---

🛠️ Tech Stack
- Python  
- Streamlit  
- MySQL  
- FLAN-T5 API (NLP Model)  

---

## ▶️ How to Run

```bash
pip install streamlit mysql-connector-python transformers
streamlit run app.py
